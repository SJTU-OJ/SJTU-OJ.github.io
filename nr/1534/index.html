    1534 卡哇伊的送分题

    题目

    Description

    琛哥喜欢玩一种单人跳棋，规则是这样的

    1. 棋盘为\(1 \times (2n+1)\)的长方形
    2. 初始状态棋盘左边\(n\)个格子里各有一个绿色的棋子，最右边\(n\)个格子里面各有一个红色棋子，正中间一个保持空格。
    3. 每一次移动，玩家可以把与空格相邻的棋子移入空格，或者把与空格间隔一个棋子的棋子移入空格，移动之后棋子原来的位置形成新的空格。
    4. 目标状态，与初始状态中心对称。
    5. 玩家的成绩为所使用的步数。

    初始状态：

    目标状态：

    作为琛哥的男票，乐乐经常被拉来和琛哥一起玩这个游戏。乐乐希望在琛哥面前表现得足够聪明所以想用最少的步数来完成，他希望你可以告诉他需要的最少步数是多少，这样他就可以根据最少步数来寻找一个合适的方案。

    Input Format

    输入为一个整数\(n\)

    Output Format

    一个整数表示最少的步数

    Sample Input 1 

    4

    Sample Output 

    24

    数据范围

    70%的数据保证答案可以用long long存储

    100%的数据保证答案小于等于\(10^{50}\)
