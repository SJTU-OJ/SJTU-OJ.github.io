    3038 棒棒糖

    题目

    Description

    N个小朋友围成一个圈，按照顺时针编号为1-N。Super Zombie Cat 想要给他们发棒棒糖。

    他每次会按照自己的心情给其中一段坐在一起的小朋友发，但是若干回合后他已经不记得

    给哪些人发过了。于是他向你寻求帮助，最后有多少个人得到了正好k颗糖？

    Input Format

    第一行一个整数N，表示小朋友个数。

    接下来一行有一个整数M，代表发糖次数。

    后面M行每行两个整数x、y(1<=x<=n，0<y<=n)，分别代表从哪个小朋友开始发并且沿顺时针给多少个小朋友发。

    最后一行一个整数K，表示询问有多少个人得到了正好K颗糖。

    Output Format

    一个整数S，代表正好得到K颗糖的人数。

    Sample Input 

    3
2
1 2
3 2
2

    Sample Output 

    1

    About Testdata 

    对于前40%的数据保证N<=1000,M<=1000

对于100%的数据保证N<=100000,M<=100000

    Limits

    Time limit: 1000ms, memory limit: 50000kb.
