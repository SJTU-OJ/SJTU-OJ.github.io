    2210 邂逅

    题目

    Description

    “邂逅”一词，出自《诗经·国风》：

    野有蔓草，零露漙兮。有美一人，清扬婉兮。邂逅相遇，适我愿兮。

    野有蔓草，零露瀼瀼。有美一人，婉如清扬。邂逅相遇，与子偕臧。

    我们的校园究竟有多大？有时候感觉好小，曾经相隔千里，如今得以在此邂逅相逢；有时候感觉好大，日夜期待，却又过尽千帆皆不是，斜晖脉脉水悠悠，肠断白蘋洲。

    科科和欢欢是一对好朋友，他们每天分别按照不同的路线在校园里面活动。他们发现，他们每天很难邂逅彼此，于是相思之苦甚是煎熬。为了能经常相遇，他们想设计一条共同的活动路线：

    假如交大有26个地点，标号分别从'a'到'z'。他们各自的路线可以表示成一个字符串，字符串上相邻两个字母表示从交大一个地点走到另一个地点。现在他们想在不改变经过各个地点先后顺序的基础上，去掉各自路线中的一些地点，使得各自剩下的路线一样。当然，剩下的路线越长越好。你的任务是帮他们找出所有这样的路线。比如，假如科科的路线是abcda，欢欢的路线是acad，那么他们最后得到的路线为：acd。

    Input Format

    第一行：一个字符串，表示科科的路线。

    第二行：一个字符串，表示欢欢的路线。

    每行字符串的长度不超过80。

    Output Format

    按照字典序输出所有满足要求的路线，每条路线一行。

    Sample Input 

    abccdac
acadc

    Sample Output 

    acac
acdc
