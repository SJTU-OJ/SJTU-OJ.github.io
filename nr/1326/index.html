    1326 texas

    题目

    Description

    德州扑克是一种很流行的纸牌游戏，现在给你两组牌，需要你比较它们的大小。

    每组牌包含5张牌，德州扑克是这样比较大小的：

    先比牌型，牌型相同者则依次比较组成牌型的牌（如都是一对则先比谁的对子大，一样大再依次比单张；都是葫芦则先比3张的，再比两张的）

    其中单张牌大小顺序为A > K > Q > J > 10 > ... > 2，另外牌型大小与具体花色无关，如黑桃皇家同花顺和方块皇家同花顺是一样大的

    牌型从大到小依次为（符合多项牌型的取最高牌型）：

    1.皇家同花顺：同花色的10,J,Q,K,A

    2.同花顺：同花色的5张连续牌

    3.四条：其中4张牌点数一样

    4.葫芦：3张相同点数的牌带2张相同点数的牌

    5.同花：5张花色相同的牌

    6.顺子：5张连续牌，如A2345，10JQKA

    7.三条：3张相同点数的牌

    8.两对：2个对子

    9.一对：1个对子

    10.高牌：不符合以上牌型的牌

    Input Format

    输入共有2n+1行。第一行一个整数n表示数据组数，之后每组两行，每行为5张牌，用空格隔开；牌用两个字符表示，前者为点数，后者为花色，其中用T代表10，花色为H(heart),S(spade),D(diamond),C(club)

    保证每组牌合法，注意德州扑克只有1副牌，不可能出现AH AH AS AD AC这样的牌型

    Output Format

    输出有n行，每行：若一样大则输出'tie'（不包含引号），否则输出A/B表示哪组牌更大

    Sample Input 

    5
TH JH QH KH AH
2S 3S 4S 5S 7D
TH JH QH KH AS
2S 3S 4S 5S 6S
TH JH QH KH AH
TS JS QS KS AS
AS AD 9C 9H 9S
9H 9S 9D 2D 9C
TH JD QC KS AC
2H 3D 4C 5S 6C

    Sample Output 

    A
B
tie
B
A

    Constraints

    对于100%的数据，n <= 1000

    共20个数据点，每个5分

    对于1~6，n = 50，仅包含高牌

    对于7~8，n = 100，仅包含高牌与一对

    对于9~10，n = 150，仅包含高牌，一对，二对

    对于11~14，n = 200，不包含顺子，同花顺，皇家同花顺

    对于15~20，200 < n <= 1000，可能包含所有牌型
