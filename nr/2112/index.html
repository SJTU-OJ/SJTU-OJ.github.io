    2112 大脸网络

    题目

    Description

    大脸平时有个爱好,就是刷Facebook.

    有一天他突发奇想：“等哥毕业了,也要搞个BigFacebook”.

    我们称它为大脸网络.

    于是,问题就来.大脸希望他开发的社交网站“Always Challenge Miracles”,所以他希望大脸网络有这么几条性质:

    1.大脸网络是个连通的网络,每位用户都可以通过直接或者间接的方式与其它用户进行数据传输;

    2.大脸网络非常特殊,每两位用户之间进行数据传输时,只有唯一的路径;

    3.大脸还有一个特别的要求,就是他的网络中的每个用户,只可以和三个用户进行直接数据传输.(当然可以不足三人)

    基于以上的三条,你需要判断一下大脸构建的网络是否符合要求.

    如果该网络符合前两条性质,那么大脸会夸你“Terrific”.

    如果该网络还符合性质三，那么大脸会夸你“Brilliant & Terrific”.

    否则大脸会说“...”.

    Input Format

    包含多组测试数据

    输入数据第\(1\)行,一个数\(T_n\),表示测试数据的组数.

    对于每组测试数据,第\(1\)行\(N,M\)表示大脸网络中的点数和边数.

    点的编号为\(1,2, \cdots ,N\).

    以后\(M\)每行两个整数\(x,y\)代表\(x,y\)之间可以进行数据直接传输.

    如果\(x,y\)出现多次，说明这两点之间存在多条可以直接数据传输的路径.

    Output Format

    对于每组测试数据,如果大脸的反应是“Terrific”,那么输出一行“t”;

    如果大脸的反应是“Brilliant & Terrific”,那么输出一行“bt”;

    否则就输出一行“.”

    Sample Input 

    2
3 2
1 2
1 3
3 2
1 2
2 1

    Sample Output 

    bt
.

    About Test Data 

    对于30%的数据,图中顶点和边的数量不超过100.

对于100%的数据,图中顶点和边的数量不超过10000.
