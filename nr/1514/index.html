    1514 拯救LHC

    题目

    Description

    大型强子对撞机（Large Hadron Collider,即LHC）是研究基础物理不可或缺的物理设备。为了避免实验受到外界干扰，LHC被决定建在地底下。

    在最近欧洲经历的难民潮中，不少伊斯兰国极端分子也混入难民队伍中潜入欧洲，并将携带的地雷埋在LHC上方 m*n 的网格中，打算用欧洲人自己的力量来炸毁LHC.

    正直勇敢爱科学的CLQ当然不会允许这样的事发生。带上自己发明的地雷探测器，CLQ踏上了保卫LHC的征程。

    然而CLQ毕竟不是专业的工程师，他发明的地雷探测器显示出来的探测结果会有一些失真。具体来说，每个格子的显示结果如下：

    • 如果该格子有雷： 3+ 以它为中心的 3*3 网格中的雷数。
    • 如果该格子没有雷： 0+ 以它为中心的 3*3 网格中的雷数。

    现在CLQ已经拿到了探测器的探测结果，请你帮助他还原出网格中的地雷分布情况。

    注意，网格之外是不存在雷的，题目保证有唯一解。

    Input Format

    第一行两个正整数 m,n 表示网格的规模。

    接下来 m 行每行 n 个数字，表示探测器的探测结果。

    Output Format

    输出 m 行长度为 n 的字符串。表示地雷分布情况。

    如果第 i 行第 j 个格子有雷，则在对应位置输出'L',否则输出'-'.

    Sample Input 

    3 4
2 6 3 2
7 5 7 5
6 7 3 2

    Sample Output 

    -L--
L-LL
LL--

    Limits

    • 对于 10% 的数据， 1≤ n, m≤ 2 ;
    • 对于 40% 的数据， 1≤ n, m≤ 4 ;
    • 对于 70% 的数据， 1≤ n, m≤ 10 ;
    • 对于 100% 的数据， 1≤ n, m≤ 1000 .
